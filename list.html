Introdução ao HTML

HTML (HyperText Markup Language) é a linguagem padrão usada para criar páginas da web. Ela permite estruturar o conteúdo e adicionar elementos como textos, imagens, links e outros. Aqui estão alguns conceitos e elementos básicos do HTML:

  1. !DOCTYPE html
    • Define o tipo de documento e a versão do HTML
  1. html
    • A tag raiz que encapsula todo o conteúdo da página HTML
  1. head
    • Contém metadados, links para arquivos de estilo, scripts, e outras informações que não são exibidas diretamente na página.
  1. title
    • Define o título da página, que é exibido na aba do navegador.
  1. meta
    • Fornece metadados sobre o documento, como a codificação de caracteres (meta charset="UTF-8") e informações de SEO.
  1. link
    • Conecta a página HTML a recursos externos, como folhas de estilo CSS (link rel="stylesheet" href="styles.css">).
  1. style
    • Contém regras de estilo CSS diretamente no documento HTML.
  1. body
    • Contém todo o conteúdo visível da página, como texto, imagens, links, etc.
  1. h1;h6
    • Tags de cabeçalho, de h1 (mais importante) a h6 (menos importante).
  1. p
    • Define um parágrafo de texto.
  1. a
    • Define um hyperlink, que pode ser usado para vincular a outra página ou a uma âncora na mesma página (a href="url">Texto do link/a).
  1. img
    • Insere uma imagem na página (img src="url_da_imagem" alt="descrição da imagem").
  1. ul;ol
    • Listas não ordenadas (ul) e ordenadas (ol). Contêm li (itens de lista).
  1. li
    • Define um item dentro de uma lista.
  1. div
    • Define uma divisão ou seção em um documento, usada para agrupar elementos e aplicar estilos.
  1. span
    • Usada para agrupar texto ou elementos inline para aplicação de estilos.
  1. form
    • Define um formulário para entrada de dados do usuário (form action="url_de_envio" method="post/get").
  1. input
    • Define um controle de entrada de dados, como campos de texto, botões de rádio, caixas de seleção, etc.
  1. button
    • Cria um botão clicável.
  1. table
    • Define uma tabela, com tr (linha da tabela), th (célula de cabeçalho), e td (célula de dados).
  1. br
    • Insere uma quebra de linha.
  1. hr
    • Insere uma linha horizontal, usada como separador de conteúdo.
  1. script
    • Inclui ou referencia scripts, geralmente JavaScript.
